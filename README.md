# Meesho-Clone

This is a fully responsive E-commerce web application built using HTML, CSS, and JavaScript. The project simulates an online shopping platform where users can browse products, view details, and add items to a cart.

 # Features
 Home page with product listings
 Multiple product categories (Men, Women, Electronics, Footwear)
 Product details page with images and description
 Add to Cart functionality using LocalStorage
 Increase / decrease product quantity
 Remove items from cart
 Automatic subtotal, tax, and total calculation
 Responsive design for mobile and desktop
